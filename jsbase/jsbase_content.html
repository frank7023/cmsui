ajax请求数据

调用方式：

cms.ajax(type, url, params, message, callback);

参数说明：

type : 请求方式 POST或 GET（必填）

url : 请求地址（必填）

params ： 请求参数，参数是一个对象，如：{params:newBook,ac1:bookinfo}

message ： 请求数据时的提示消息,可以是字符串，空字符串''，null。如果为null，则请求没有toast弹窗

callback ： 回调函数，返回请求到的数据及所传的参数params

使用方法如下： 
                        
cms.ajax(
    'GET', 
    'http://***', 
    {param1:param1,param2:param2}, 
    '这里可以为空字符串',//如果为空字符串''，toast提示窗显示默认的“加载中，请稍后...” 
    function(data,params){
    //data为请求到的数据
    //params为回传的请求参数
});

//或则将回调与在外面。
cms.ajax(
    'GET', 
    'http://***', 
    {param1:param1,param2:param2}, 
    null, //没有toast弹窗 
    fun
   );
   
   function fun(data,params){
       //处理自己的逻辑
   }

窗口之间的通讯

调用方式：

1、通知要执行js的窗口：appcan.window.publish('functionName', "params");

2、接收执行通知的窗口：appcan.window.subscribe('functionName', function(msg){});

参数说明：

functionName执行通知窗口要执行的js方法名，params为要传入的参数，msg为接传来的参数（就是params）。

注意：

appcan.window.subscribe要写在appcan.ready(function(){})中，且functionName要一致。

详情请参考appcan官方文档。查看

     appcan.window.publish("notice", "这段文字是从主窗口传过来的哦！");

可输入提示框

调用方式：

appcan.window.prompt(title,content, defaultValue,buttons,callback)

参数说明：

title:提示框的标题

content:提示框中显示的内容

defaultValue:(String类型) 必选 输入框默认文字

HTML5 Web 存储包括保存数据，取出数据及清空数据。

详情请参考HTML5 Web 存储文档。查看

1、保存数据。

调用方式：

cms.setLocVal(key, Val)

参数说明：

key:要保存的键。

val:要保存对应的值。

保存键为“locStorageTest1”，值为”如果不清空，该条数据将永远存在“的数据。保存


保存键为“locStorageTest2”，值为”您好！“的数据。保存

可以在下面获取保存的数据取值。

2、获取保存的数据。

只要不清空数据，可以在任何页面取得已保存的数据。

调用方式：

cms.getLocVal(key)

参数说明：

key:要获取值的键。

获取键为locStorageTest2的值，试试

3、清除保存的数据。

调用方式：

cms.removeLocVal(key)

参数说明：

key:要清除的键。key可以为一个字符串，可以为一个数组，可以为空。

注意：

为空时清除所有的本地存储，要慎重始用。由于安卓有返回键，所以需在退出的时候调用cms.listenBack(key)。

清除键为locStorageTest1的值，点击消除后就不能再获取到locStorageTest1的值了。


key为数组时清除多个值，点击消除所有的本地存储都被清除了。


清除所有值，点击消除所有的本地存储都被清除了。 

                        
//清除键为locStorageTest1的值
cms.removeLocVal('locStorageTest');
//key为数组时清除多个值
cms.removeLocVal(['locStorageTest1','locStorageTest2']);
//清除所有值
cms.removeLocVal();

日期格式转换，输出指定格式

调用方式：

cms.strToDate(dateStr,type, separate)

参数说明：

dateStr:传入的时间。格式类型如下： “2015-11-05 09:34:25”、 “2015/11/05 09:34:25”、 “2015年11月5日 19:41:44”、 1472639579（时间戳）

type:返回的时间。 y:返回年、 md:返回月日、 ymd：返回年月日、 hi:返回时分、 his：返回时分秒、 默认全部返回。

separate:返回的时间格式分隔符。 特殊：分隔符为 ch时返回时间类型如：2016年08月09日

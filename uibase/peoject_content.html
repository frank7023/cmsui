文档概要和设置

文档类型

AppCan开发平台是基于HTML5+CSS3+JavaScript技术的跨平台移动应用。项目中的页面采用标准的HTML5头部声明。<title>标签可有可无，留空也可。接下来是声明页面编码，建议使用UTF8编码，因为JS中ajax与json编码只能是UTF8，避免出现乱码和程序异常。 
                        
<!DOCTYPE html>
<html class="um landscape min-width-240px min-width-320px min-width-480px min-width-768px min-width-1024px">
<head>
    <title></title>
    <meta charset="utf-8">
</head>
</html>

页面设置

viewport元属性的设置，这里分别解释一下content里的内容。

target-densitydpi=device-dpi，指定屏幕像素密度DPI，device-dpi 为设备原本的DPI值，不会有任何缩放。

width=device-width，指定屏幕宽度，device-width设备屏幕的宽度。

initial-scale=1，初始缩放比例，设置为1，表示不进行任何缩放，显示原始大小。

user-scalable=no，用户是否可以手动缩放页面，设为no不允许。

minimum-scale=1.0，最小缩放比例，设为1，最小只能缩放到1倍。

maximum-scale=1.0，最大缩放比例，设为1，最大只能缩放到1倍，与上一项配合，保证，页面不会进行缩放。 

                    
<meta name="viewport" content="target-densitydpi=device-dpi, width=device-width, initial-scale=1, user-scalable=no, minimum-scale=1.0, maximum-scale=1.0">
这样设置完成后，就能够保证，在任何设备中，网页的宽度就是屏幕显示区域的宽度，不会发生变化，这样我们进行排版里就不会出现问题，这也是与普通web开发比较大的不同之处了。

页面结构和模板

页面头(命名格式：xxx.html)模板

app页面有两部分组成，头文件和内容文件。头文件是app界面的标题栏部分，可定义返回键、标题、自定义功能按钮。 
                        
<!DOCTYPE html>
<html class="um landscape min-width-240px min-width-320px min-width-480px min-width-768px min-width-1024px">
    <head>
        <title></title>
        <meta charset="utf-8">
        <meta name="viewport" content="target-densitydpi=device-dpi, width=device-width, initial-scale=1, user-scalable=no, minimum-scale=1.0, maximum-scale=1.0">
        <link rel="stylesheet" href="css/ui-base.css">
    </head>
    <body class="um-vp bc-bg">
        <div class="up ub ub-ver" tabindex="0">
            <!--header开始-->
            <div id="header" class="uh bc-text-head ub bc-head">
                <div class="ub ub-ac uinn-rl04 umw3" id="nav-left">
                    <!--回退键-->
                </div>
                <h1 class="ut ub-f1 ulev-nav ut-s tx-c" tabindex="0">标题</h1>
                <div class="ub ub-ac umw3" id="nav-right">
                    <!--自定义功能按钮区-->
                </div>
            </div>
            <!--header结束--><!--content开始-->
            <div id="content" class="ub-f1 tx-l">
                    <!--主功能区-->
            </div>
            <!--content结束-->
        </div>
        <script src="js/appcan.js"></script>
        <script src="js/appcan.control.js"></script>
    </body>
    <script>
        appcan.ready(function() {
            //加载xxx_content.html
        });        
    </script>
</html>

页面内容(命名格式：xxx_content.html)模板

内容文件是app界面的主功能区，在头文件中加载。 
                        
<!DOCTYPE html>
<html class="um landscape min-width-240px min-width-320px min-width-480px min-width-768px min-width-1024px">
    <head>
        <title></title>
        <meta charset="utf-8">
        <meta name="viewport" content="target-densitydpi=device-dpi, width=device-width, initial-scale=1, user-scalable=no, minimum-scale=1.0, maximum-scale=1.0">
        <link rel="stylesheet" href="css/ui-base.css">
    </head>
    <body class="um-vp bc-bg">
        <!--编写页面区域-->
        <script src="js/appcan.js"></script>
        <script src="js/appcan.control.js"></script>
    </body>
    <script>
        //js函数
    </script>
</html>
